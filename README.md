# 📚 Cbamo

`기초`부터 시작하는 CS 스터디입니다. `프론트엔드, 백엔드 관련 지식` 및 `CS 기초 역량 강화`가 목표입니다.

`2025.01.17 ~`

[`Cbamo Wiki`](https://github.com/scorchedrice/cbamo/wiki)

### Ground Rule

- 주 2회 `화요일/ 금요일` 진행
  - 스터디 결석해야 할 경우, `✅사전에` 스터디원들에게 공지할 것
- 한 주마다 `🙋‍♀️스터디원`이 돌아가며 대주제를 선정 (프론트엔드, 백엔드, 데이터, 컴퓨터구조 ...)
  - 주제는 해당 `📄리드미` 하단에 추가
